一套系統，兩種使用情境

老師上課可投影進行分隊搶答；學生也能用手機或平板自主練習、累積錯題。

380 筆單字與片語 支援 1上 / 1下 / Lesson 篩選 免登入、可離線開啟

老師怎麼用

  1. 選擇題庫範圍與出題方向。
  2. 投影後按「開始比賽」。
  3. 學生搶答後按空白鍵顯示答案，再幫隊伍加分。

學生怎麼用

  1. 掃 QR Code 或開啟網址。
  2. 選擇 1上、1下或指定 Lesson。
  3. 看題目、顯示答案，將不熟的字加入錯題。

建議發佈

把整個資料夾上傳到 GitHub Pages，產生固定網址；正式比賽時也保留單檔 HTML 備份。
